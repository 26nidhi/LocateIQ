# TrackBack - Secure Lost and Found Application

TrackBack is a comprehensive secure lost-and-found web application built using the MERN stack (MongoDB, Express, React, Node.js) with real-time chat and notifications powered by Socket.IO. It supports community-based lost and found item reporting, role-based access control, owner approval workflows, and item matching with notifications.

---

## Features

- User authentication with JWT
- Role-based access control: Admin, CommunityOwner, User
- Community creation and management
- Lost and found item reporting with location and text matching
- Real-time notifications on matched items
- Private real-time chat between matched users
- Owner approval workflow for community ownership
- Comprehensive input validation and error handling
- RESTful API backend with MongoDB
- React frontend with React Router and Context API
- Responsive UI with Tailwind CSS
- Unit and integration tests for backend and frontend

---

## Tech Stack

- Backend: Node.js, Express, MongoDB, Mongoose, JWT, Socket.IO, Joi
- Frontend: React, React Router, Axios, Socket.IO Client, Tailwind CSS
- Testing: Jest, Supertest, React Testing Library
- Deployment-ready configuration with environment variables

---

## Prerequisites

- Node.js (v18 or newer recommended)
- npm (comes with Node.js)
- MongoDB (local or remote instance)
- Git (optional, for cloning repository)
